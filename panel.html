翻译悬浮窗设置

输入你的自定义翻译服务的完整 API 端点。
这个服务应该能接收一个 JSON 请求体，例如： 

{ "text": "要翻译的文本", "target_lang": "zh" }
并返回一个 JSON 响应，例如： 
{ "translatedText": "翻译后的文本" }

重要： 你的自定义 API 必须正确配置 CORS，允许 SillyTavern 页面进行跨域请求。

如果你的自定义翻译 API 需要认证密钥，请在此处输入。
安全警告： 直接在客户端（浏览器）存储和使用 API 密钥存在安全风险。
更安全的做法是通过你自己的后端代理服务器来处理翻译请求，并将 API 密钥存储在服务器端。
