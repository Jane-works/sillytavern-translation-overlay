安装说明

要将此插件添加到SillyTavern，请按照以下步骤操作：

  1. 在SillyTavern中，转到扩展管理页面
  2. 点击"添加第三方扩展"
  3. 输入以下URL: https://raw.githubusercontent.com/yourusername/ai-translator-plugin/main/plugin.json
  4. 点击"安装"并重启SillyTavern

使用方法

1. 在API设置部分配置您的API端点、密钥和模型

2. 选择您希望翻译成的语言

3. 点击"保存设置"和"测试连接"确保配置正确

4. 当AI回复包含<char>标签时，内容将自动被捕获并翻译

示例AI回复:

<char>Hello, how are you today?</char> I hope you're having a wonderful day.

将捕获"Hello, how are you today?"并翻译为所选语言。

支持的AI API

此插件支持以下AI提供商:

  • OpenAI (GPT-3.5, GPT-4)
  • Anthropic (Claude系列)
  • 自定义API端点 (兼容OpenAI格式)
